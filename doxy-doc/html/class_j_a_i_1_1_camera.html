Detailed Description

Class implement JAI Camera interface.

Example of usage: JAI::Camera *camera = JAI::Camera::getCameraList().front(); camera.open(); //check return value camera.start(); //check return value //in loop, with try and catch blocks in case of getting new image, when camera haven't return it yet. cv::Mat image = camera.getNextFrame(); //do stuff with image camera.stop(); camera.close();
